# 招聘 AI Demo 产品需求

## 核心痛点

HR 在企业微信、群聊、腾讯会议和腾讯文档之间手动记录招聘数据，存在录入慢、遗漏多、更新滞后、复盘难的问题。

## MVP 目标

以最低成本完成一个可运行 Demo，让 HR 能自动沉淀候选人信息、面试表现、评分、状态和岗位汇总报告。

## P0 范围

- 简历上传与解析
- 候选人入库与列表
- 面试纪要分析
- 面试官评分
- 候选人状态更新
- 招聘数据看板
- 单岗位 10 名候选人汇总报告
- 企业微信群同步文案

## 非目标

- 当前不接企业微信机器人
- 当前不接腾讯文档 API
- 当前不实现权限系统
- 当前不部署本地私有模型

## 演示路径

1. 启动后端和前端。
2. 点击“报告同步”。
3. 点击“生成 10 人演示数据”。
4. 查看岗位 Markdown 汇总报告。
5. 查看“数据看板”。
6. 生成群同步文案。
