# P0 Demo 开发计划

## 目标

完成一个可演示的 AI 招聘提效 Demo，覆盖简历录入、候选人管理、面试分析、数据看板、岗位 10 人汇总报告和群同步文案。

## 已完成

- React + Vite 前端基础页面
- FastAPI 后端接口
- Conda 环境配置
- SQLite 数据表初始化
- Mock AI 简历解析
- Mock AI 面试分析
- 候选人入库与列表
- 面试记录保存与状态更新
- ECharts 数据看板
- 岗位 10 人 Markdown 汇总报告
- 企业微信群同步文案
- 一键生成 10 人演示数据
- 面试者投递申请页
- `/apply` 独立投递入口
- HR 后台不展示投递跳转按钮
- PDF 简历解析后自动回填申请信息
- 候选人列表按在招岗位筛选
- 候选人 CSV 导出入口
- esbuild 生产构建脚本
- GitHub 远程仓库绑定与推送

## 待完成

- 增加真实模型 API 配置开关
- 完善 README 演示流程

## 验收标准

- 后端可通过 Conda 环境启动
- 前端可通过 `npm run dev` 启动
- 点击“生成 10 人演示数据”后可看到岗位汇总报告
- 面试者可通过 `/apply` 上传 PDF 简历并提交申请
- 候选人列表可按在招岗位筛选
- 数据看板展示候选人总数、平均分和分布图表
- 群同步文案可生成并复制
- 候选人数据可导出 CSV
- 前端可通过 `npm run build:esbuild` 生成生产产物
