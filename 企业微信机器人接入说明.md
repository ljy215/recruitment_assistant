# 企业微信机器人接入说明（后续扩展）

当前 Demo 阶段暂不接入企业微信机器人，因为需要企业或学校管理员开放群机器人权限。本文件仅作为后续企业开通权限后的扩展参考，当前方案以“生成群消息文案 + 手动复制到企业微信群”为主。

## 1. 创建群机器人

1. 打开企业微信群。
2. 点击右上角群设置。
3. 找到“群机器人”。
4. 添加机器人，名称可设置为“招聘助手”。
5. 复制机器人 Webhook 地址。

Webhook 示例格式：

```text
https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=xxxxxxxx
```

注意：Webhook 等同于机器人密钥，不要提交到公开代码仓库。

## 2. 发送测试消息

在当前目录运行：

```powershell
python .\wecom_bot.py --webhook "你的企业微信机器人Webhook地址"
```

如果想自定义候选人信息：

```powershell
python .\wecom_bot.py --webhook "你的企业微信机器人Webhook地址" --candidate "李四" --position "产品经理" --status "终面通过" --score "91" --summary "业务理解较好，表达清晰，岗位匹配度高。" --next-action "进入薪资沟通"
```

## 3. 在 Demo 代码中调用

```python
from wecom_bot import build_recruiting_status_message, send_text_message

webhook_url = "你的企业微信机器人Webhook地址"

message = build_recruiting_status_message(
    candidate_name="张三",
    position="售前解决方案顾问",
    status="一面通过，待终面",
    score="88",
    summary="表达清晰，项目经验匹配度较高，技术深度需终面确认。",
    next_action="安排业务负责人终面",
)

send_text_message(webhook_url, message)
```

## 4. 推荐推送内容

群消息建议只推送招聘状态摘要：

- 岗位
- 候选人姓名
- 当前状态
- 综合评分
- 面试摘要
- 下一步动作

不要在群消息中展示完整手机号、邮箱、身份证号等敏感信息。
